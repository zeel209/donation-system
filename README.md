# Unity Foundation - Online Charity & Donation System

Unity Foundation is a web-based charity and donation management system developed using PHP and MySQL.

## Features

- User Registration & Login
- Secure Donation System
- UPI Payment
- Payment Verification
- Donation Receipt
- Volunteer Registration
- Admin Dashboard
- Campaign Management
- Contact Form
- Responsive Design

## Technologies Used

- PHP
- MySQL
- HTML5
- CSS3
- JavaScript
- Bootstrap
- XAMPP

## Installation

1. Clone the repository

```bash
git clone https://github.com/zeel209/donation-system.git
```

2. Copy the project into:

```
C:\xampp\htdocs\
```

3. Import the database using phpMyAdmin.

4. Update database credentials in `db.php`.

5. Start Apache and MySQL.

6. Open:

```
http://localhost/unity foundation
```
